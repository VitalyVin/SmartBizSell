# Настройка Яндекс.Вебмастер для SmartBizSell.ru

## Что такое Яндекс.Вебмастер

Яндекс.Вебмастер — это бесплатный инструмент от Яндекса для вебмастеров, который помогает отслеживать индексацию сайта в поисковой системе Яндекс, позиции, ошибки и производительность.

## Шаги настройки

### 1. Регистрация в Яндекс.Вебмастер

1. Перейдите на https://webmaster.yandex.ru
2. Войдите в аккаунт Яндекс (или создайте новый)
3. Нажмите "Добавить сайт"

### 2. Добавление сайта

1. Введите URL вашего сайта: `https://smartbizsell.ru`
2. Нажмите "Добавить"

### 3. Верификация сайта

Яндекс предложит несколько способов верификации. Рекомендуемый способ:

#### Метод 1: HTML-файл (самый простой)

1. Скачайте HTML-файл, который предложит Яндекс
2. Загрузите его в корневую директорию сайта через FTP/SSH
3. Убедитесь, что файл доступен по адресу: `https://smartbizsell.ru/yandex_XXXXX.html`
4. Нажмите "Проверить" в Яндекс.Вебмастер

#### Метод 2: Мета-тег (альтернативный)

1. Выберите метод "Мета-тег"
2. Скопируйте предоставленный мета-тег
3. Добавьте его в секцию `<head>` файла `index.php`
4. Нажмите "Проверить"

#### Метод 3: DNS-запись (для продвинутых)

1. Выберите метод "DNS-запись"
2. Добавьте TXT-запись в DNS вашего домена
3. Дождитесь распространения DNS (может занять до 48 часов)
4. Нажмите "Проверить"

### 4. Отправка sitemap

После успешной верификации:

1. В левом меню выберите **Индексирование** → **Файлы Sitemap**
2. В поле "URL файла Sitemap" введите: `https://smartbizsell.ru/sitemap.php`
3. Нажмите "Добавить"
4. Яндекс начнет индексировать ваш сайт

### 5. Настройка уведомлений

1. Перейдите в **Настройки** → **Уведомления**
2. Включите уведомления по email о важных событиях
3. Укажите email для получения уведомлений

### 6. Настройка robots.txt

Яндекс автоматически проверит ваш `robots.txt`. Убедитесь, что файл доступен по адресу:
`https://smartbizsell.ru/robots.txt`

## Что дальше?

### Мониторинг индексации

- **Страницы в индексе**: проверяйте, сколько страниц проиндексировано
- **История индексации**: отслеживайте динамику индексации
- **Исключенные страницы**: проверяйте, какие страницы исключены из индекса

### Анализ поисковых запросов

- **Запросы**: отслеживайте, по каким запросам ваш сайт показывается в Яндексе
- **Позиции**: смотрите позиции вашего сайта по ключевым запросам
- **CTR**: анализируйте кликабельность в поисковой выдаче

### Исправление ошибок

- **Ошибки индексирования**: исправляйте проблемы с доступностью страниц
- **Проблемы безопасности**: получайте уведомления о взломе
- **Мануальные фильтры**: исправляйте проблемы, которые могут влиять на ранжирование

### Дополнительные инструменты

- **Оригинальные тексты**: отправляйте оригинальный контент для защиты от копирования
- **Переезд сайта**: используйте при смене домена
- **Геозависимость**: настройте регион для вашего сайта (Россия)

## Важные моменты

1. **Верификация может занять время**: после добавления файла или тега может потребоваться несколько минут для проверки
2. **Индексация не мгновенна**: Яндекс может занять несколько дней или недель для полной индексации сайта
3. **Регулярно проверяйте**: заходите в Яндекс.Вебмастер раз в неделю для мониторинга статуса
4. **Исправляйте ошибки**: быстрое исправление проблем улучшает ранжирование
5. **Учитывайте регион**: для российского сайта Яндекс особенно важен, так как это основная поисковая система в России

## Особенности для российских сайтов

- Яндекс является основной поисковой системой в России
- Важно правильно настроить регион сайта
- Используйте кириллические ключевые слова
- Учитывайте особенности русского языка в SEO
