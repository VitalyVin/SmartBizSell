# Настройка Google Search Console для SmartBizSell.ru

## Что такое Google Search Console

Google Search Console (ранее Google Webmaster Tools) — это бесплатный инструмент от Google, который помогает отслеживать индексацию сайта, позиции в поиске, ошибки и производительность.

## Шаги настройки

### 1. Регистрация в Google Search Console

1. Перейдите на https://search.google.com/search-console
2. Войдите в аккаунт Google (или создайте новый)
3. Нажмите "Добавить ресурс"

### 2. Добавление сайта

1. Выберите тип ресурса: **Префикс URL**
2. Введите URL вашего сайта: `https://smartbizsell.ru`
3. Нажмите "Продолжить"

### 3. Верификация сайта

Google предложит несколько способов верификации. Рекомендуемый способ:

#### Метод 1: HTML-файл (самый простой)

1. Скачайте HTML-файл, который предложит Google
2. Загрузите его в корневую директорию сайта через FTP/SSH
3. Убедитесь, что файл доступен по адресу: `https://smartbizsell.ru/googleXXXXX.html`
4. Нажмите "Проверить" в Google Search Console

#### Метод 2: HTML-тег (альтернативный)

1. Выберите метод "HTML-тег"
2. Скопируйте предоставленный мета-тег
3. Добавьте его в секцию `<head>` файла `index.php`
4. Нажмите "Проверить"

#### Метод 3: DNS-запись (для продвинутых)

1. Выберите метод "DNS-запись"
2. Добавьте TXT-запись в DNS вашего домена
3. Дождитесь распространения DNS (может занять до 48 часов)
4. Нажмите "Проверить"

### 4. Отправка sitemap

После успешной верификации:

1. В левом меню выберите **Sitemaps**
2. В поле "Добавить новую карту сайта" введите: `https://smartbizsell.ru/sitemap.php`
3. Нажмите "Отправить"
4. Google начнет индексировать ваш сайт

### 5. Настройка уведомлений

1. Перейдите в **Настройки** → **Пользователи и разрешения**
2. Добавьте email для получения уведомлений о проблемах с сайтом
3. Включите уведомления по email

## Что дальше?

### Мониторинг индексации

- **Покрытие**: проверяйте, какие страницы проиндексированы
- **Sitemaps**: следите за статусом отправленного sitemap
- **Проверка URL**: используйте инструмент для проверки индексации конкретных страниц

### Анализ производительности

- **Производительность**: отслеживайте позиции в поиске, CTR, количество показов
- **Улучшения**: получайте рекомендации по оптимизации
- **Мобильная удобность**: проверяйте, как сайт отображается на мобильных устройствах

### Исправление ошибок

- **Ошибки индексирования**: исправляйте проблемы с доступностью страниц
- **Проблемы безопасности**: получайте уведомления о взломе или вредоносном ПО
- **Мануальные действия**: исправляйте проблемы, которые могут влиять на ранжирование

## Важные моменты

1. **Верификация может занять время**: после добавления файла или тега может потребоваться несколько минут для проверки
2. **Индексация не мгновенна**: Google может занять несколько дней или недель для полной индексации сайта
3. **Регулярно проверяйте**: заходите в Search Console раз в неделю для мониторинга статуса
4. **Исправляйте ошибки**: быстрое исправление проблем улучшает ранжирование
